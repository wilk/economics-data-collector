# Economics Data Collector
A personal economics data collector

## Requirements

 - docker (>= 17.06.0-ce)
 - docker-compose (>= 1.14.0)

## Install
Few simple steps:

1. clone
```bash
$ git clone https://github.com/wilk/economics-data-collector.git
```

2. setup
```bash
$ docker-compose run --rm setup
```
